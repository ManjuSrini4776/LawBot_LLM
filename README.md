Project Overview

LawBot is an AI-powered Retrieval Augmented Generation (RAG) system
designed to answer legal questions based on internal case documents.
It uses embeddings, vector search, and an LLM to provide accurate,
source-based answers.



Key Features

Upload legal documents (PDF)
Extract and convert into vector embeddings
Semantic search over large case database
LLM-based answer generation
Provides cited references


Tech Stack

Python, LangChain, FAISS/ChromaDB, HuggingFace, Streamlit
